# Let's Travel
사용자의 위치를 기반으로 여행 장소, 경로 설정을 해주며, 사용자가 여행지를 커스텀하여 저장할 수 있도록 도와주는 어플리케이션

---

- Google Api Maps 사용
- 검색
- 사용자 주변 장소 추천
- 길찾기 제공
- 사용자 목록 커스텀
  
📌 Bottom Navigation
  - 주변
    - [x] 식당, 숙소, 버스 등 장소 검색바 (AutocompleteSupportFragment 사용)
    - [x] 사용자 위치 기반으로 주변 장소 띄우기
    - [x] 장소 검색 후 마크 표시 및 사진/장소 이름/주소 띄우기
    - [ ] 장소 즐겨찾기
